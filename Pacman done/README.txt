თუ თამაშის გაშვება გსურთ Shortcut-ით (Pacman), აუცილებელია პაკმენის ფაილი (Pacman done) იყოს Desktop-ზე,
ამის შემდეგ კი ნებისმიერ ადგილას შეიძლება დაყენება Shortcut-ის.
If you want to turn game on, you must transfer main file (Pacman done) on the desktop,
after this you can transfer game icon or shortcut anywhere you want.
